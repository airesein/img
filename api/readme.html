图片交互

  • 鼠标左键单击图片: 可进行图片预览（在大图模式下，再次单击大图或按ESC键可退出预览）。
  • 鼠标右键单击图片: 复制该图片的公链地址到剪贴板。

随机图片 API 调用方式

API 地址：https://t.ziworld.top/api

1. 随机获取单个分类图片

通过在 URL 中添加 category 参数指定单个分类名称。

示例：

https://t.ziworld.top/api?category=原神

2. 随机获取多个分类图片

通过在 URL 中使用逗号 , 分隔多个分类名称。

示例：

https://t.ziworld.top/api?category=原神,PC,二次元

3. 不指定分类

如果 URL 中不包含 category 或 apicategory 参数，将使用默认的 PC 分类随机获取图片。

示例：

https://t.ziworld.top/api?

重要提示： 为了避免 URL 解析问题，特别是分类名称中包含特殊字符（如中文、空格等），强烈建议使用 **URL 编码** 的形式传入分类参数。

此时，请使用 apicategory 作为参数名，而不是 category

示例： 原神 经过 URL 编码后为 %E5%8E%9F%E7%A5%9E

https://t.ziworld.top/api?apicategory=%E5%8E%9F%E7%A5%9E
